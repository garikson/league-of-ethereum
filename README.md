# dot-boardroom
BoardRoom dApp (built with dot-dom)

## Install

```
git clone http://github.com/boardroom-project/dot-boardroom
cd dot-boardroom

npm install
```

## Start Server

```
npm start
```

## Build

```
npm run build
```

## Contracts

```
npm run build:contracts
npm run deploy:ropsten
```

View surge instructions in `package.json` for proper surgyness..
